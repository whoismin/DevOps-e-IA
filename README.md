# Algoritmo de Temperatura com IA em Python

## Descrição

Este projeto tem como objetivo desenvolver um programa em Python que utiliza uma API para detectar a temperatura do local e, com base nessas informações, indicar qual tipo de roupa é mais favorável para o dia.

A atividade faz parte de um estudo sobre Inteligência Artificial, aplicando regras simples de decisão para analisar a temperatura e sugerir roupas adequadas ao clima.

## Objetivo da Atividade

O objetivo da atividade é criar um programa com API em Python capaz de consultar a temperatura atual de um local e informar ao usuário qual roupa é mais recomendada para o dia.

## Tecnologias Utilizadas

- Python
- Lógica de decisão/IA simples

## Funcionalidades

- Consultar a temperatura atual do local
- Identificar se o clima está frio, ameno ou quente
- Recomendar roupas adequadas para o dia

## Exemplo de Funcionamento

Se a temperatura estiver baixa, o programa pode sugerir o uso de blusa de frio, casaco ou calça.

Se a temperatura estiver amena, pode recomendar roupas leves com uma jaqueta.

Se a temperatura estiver alta, pode sugerir camiseta, shorts, vestido ou roupas mais frescas.
